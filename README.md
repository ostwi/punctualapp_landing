# punctual-app.com

Landing page for **Punctual** — a free, open-source macOS menu bar app that makes sure you're never late to a meeting.

Live at [punctual-app.com](https://punctual-app.com).

## How it works

This is a plain static site (no build step). Hosting is **Cloudflare Pages**, connected to this GitHub repo — push to `main` and the site deploys automatically within a minute.

Pages:

- `/` — landing page (`index.html`)
- `/privacy/` — privacy policy (required for Google OAuth verification)
